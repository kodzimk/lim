LIM
---------------------------------------------------
c library that works with limis(math)
</pre>
</div>

## Installation

install this repo.

```sh
git clone (link of project)
make 
```

## Usage example
```sh
lim n-> 0 : n + 1
lim n-> 0 : n + 1 * ( 2 + 2)
lim n-> 0 : n + 1 * ( 2 + 2) | 2 * 2^2
```
